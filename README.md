# RingTimeSelectView
A multi-selector for time between 0 and 60 minute
一个分钟时间段的选择器，可选择多个

## 操作演示：
![demoGif](https://github.com/zjun615/RingTimeSelectView/blob/master/img/demo.gif)

## 主要功能:
 - 在0~60之间，选择一个或多个时间段
 - 添加：点击圆环空白处添加时间段
 - 删除：起始点和终止点在同一时间点
 - 修改：滑动起始点或终止点
